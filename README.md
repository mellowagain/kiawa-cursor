# How to use

1. Download the `kiawa.avi` file
2. Press `Download`
3. Go to Windows Cursor Settings (Search "Mouse" -> Right Side -> Additional mouse options)
4. Press `Pointers`
5. Select `Browse` on the standard cursor and put select the `kiawa.avi` file
